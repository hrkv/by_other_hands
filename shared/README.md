Общие модули (`shared/`)
========================

Назначение: единый источник правды для правил игры и общих типов данных, используемых и клиентом, и сервером.

### Черновая структура

- `src/`
  - `types/` — общие типы/интерфейсы (игрок, клетка, матч, сообщения сети и т.п.).
  - `rules/` — логика core loop (ходы, эффекты бустов, проверка победы).

Планируется, что игровые правила здесь будут максимально “чистыми” (без зависимостей от графики и сети), чтобы проще было тестировать и портировать игру.
