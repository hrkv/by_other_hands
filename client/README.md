Клиентская часть (`client/`)
============================

Назначение: веб‑клиент игры (рендер поля, ввод, UI, подключение к серверу).

### Черновая структура

- `src/`
  - `app/` — инициализация приложения, корневой UI.
  - `game/` — визуализация игрового поля и сущностей.
  - `net/` — сетевое взаимодействие с сервером.
  - `state/` — состояние клиента (очередь хода, локальные настройки и т.п.).
- `public/`
  - `index.html` — точка входа для браузера.

Конкретный стек (Phaser / PixiJS / Godot Web export / др.) выбирается на следующем шаге; структура организована так, чтобы отделять модель игры (в `shared/`) от визуализации.
