Ассеты (`assets/`)
==================

Назначение: все внешние ресурсы игры.

### Предлагаемая структура

- `sprites/` — спрайты, тайлсеты, иконки.
- `audio/` — звуковые эффекты, музыка/эмбиент.
- `levels/` — описания арен/полей (JSON/другой формат данных).

Конкретные форматы и ограничения (разрешения, размер файлов и т.п.) будут зафиксированы позже в рамках минимального ассет‑pipeline.
