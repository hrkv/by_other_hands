Серверная часть (`server/`)
===========================

Назначение: управление матчами, матчмейкинг, валидация ходов и синхронизация состояния игры между двумя клиентами.

### Черновая структура

- `src/`
  - `index.ts` — точка входа сервера.
  - `match/` — логика матчей, состояние партии.
  - `net/` — сеть/транспорт (WebSocket/HTTP, протокол сообщений).

Конкретный стек (Node/WebSocket, другой бэкенд и т.п.) выбирается после подтверждения архитектуры.  
Правила игры и структуры данных желательно выносить в `shared/`, чтобы их переиспользовать на клиенте.
